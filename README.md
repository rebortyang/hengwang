README 
恒旺电器网站设计
主要内容如下：
1. 店面网站：店面资料、导航、主要商品、商品列表、检索
2. 仓库管理系统：商品、进库、出库、权限控制

开发环境：
* 语言：python （主体）、js+css（样式、操作）
* 框架：django 、sanic（flask）
* 工具：pycharm 、sublime
* 部署环境：阿里云

开发流程：
1. 需求分析：交付需求分档，内容有功能模块和基本界面；
2. 概要设计：包括系统的基本处理流程、系统的组织结构、模块划分、功能分配、接口设计、运行设计、数据结构设计和出错处理设计；
3. 详细设计：描述实现具体模块所涉及到的主要算法、数据结构、类的层次结构及调用关系，能够根据详细设计报告进行编码；
4. 编码：根据《详细设计报告》进行程序编码；
5. 测试

开发方法：
测试驱动（unittest + selenium + django测试包提供内容）
